## Requirements

- Python 3.x
- Jupyter Notebook
- Required libraries: NumPy, Pandas, Matplotlib, Seaborn (for visualization in lab `exp_18.ipynb`)

## How to Use

1. Clone the repository:
   ```bash
   git clone https://github.com/anshgirigoswami-sudo/MCA_Python_Programming_for_problem_solving.git
   ```

2. Navigate to the desired directory:
   ```bash
   cd Ansh/Python/Lab\ assisment/
   # or
   cd Ansh/Python/Pratical\ record/
   ```

3. Open Jupyter notebooks:
   ```bash
   jupyter notebook exp_1.ipynb
   ```

4. Run Python files:
   ```bash
   python main.py
   ```
